API

Supported Response Types

You can use the 

Accept
header to dictate the response type.

Parts Horse currently understands the following 

Accept
headers:

Accept
header		 Type Suitable for... 
text/plain
 Plain-text Dumping to a terminal with cURL. 
text/html
 HTML Web browsers. 
application/json
 JSON Programmatic access.

Endpoints

Endpoint Purpose
/search?q=:query Search for :query and return all matches.
/parts A list of all documented parts.
/parts/:part Information for :part.
/datasheets/:part (alias: /ds/:part) Redirects to the datasheet for :part. This URL is included in /parts/:part responses.
